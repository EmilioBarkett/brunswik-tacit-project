# German Credit Dataset Decoder

Decodes the German Credit Dataset (german.data) from encoded categorical
values into a human-readable CSV file.

## Files
- `german.data` — raw dataset (1000 rows, 21 attributes)
- `decode_german_credit.py` — decoding script
- `german_credit_decoded.csv` — output (generated by the script)

## How to run
```bash
python decode_german_credit.py
```

## Output
Produces `german_credit_decoded.csv` with plain-English labels for all
categorical attributes and a `credit_risk` column (Good/Bad).
